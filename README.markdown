## Seacrest

* http://wiki.github.com/panpainter/seacrest

### DESCRIPTION:

A collection of tools that helps webmasters, developers, and designers maintain a neat and tidy codebase.

### FEATURES/PROBLEMS:

Build the following

- Identify broken URLs (internal, external and resources)
